# Hasan Öngen Barber App

A modern barber shop management application built with **Next.js**, **React**, **TypeScript**, **Firebase**, and **Capacitor**.

This repository is a **showcase version** of the original project. Some business-specific logic, backend services, and sensitive configurations have been intentionally removed.

---

## Features

- Appointment Management
- Customer Management
- Employee Management
- Service Management
- Store Settings
- Push Notifications
- Mobile Ready (Android & iOS)
- Progressive Web App (PWA)
- Responsive User Interface
- Firebase Authentication
- Cloud Firestore Integration

---

## Running Locally

```bash
npm install

npm run dev
```

Production

```bash
npm run build
npm start
```

---

## Mobile

The application supports Android and iOS using Capacitor.

```bash
npx cap sync
npx cap open android
```

or

```bash
npx cap open ios
```
